## Environment:

**Required Libraries**

- openssl 3.0.2 15

- PBC     0.5.4

## How to compile:
After you install the required libraries, run the command below to compile the program.

``` cmake
sh ./compile.sh
```

Then the program will appear in the `bin` folder.
